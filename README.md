# Скрипт мониторинга сайтов

Скрипт на вход получет текстовый файл в котором указаны ссылки для проверки сайтов.

Для запуска скрипта требуется выполнить команду:
```{r, engine='bash'}

$ python check_site_health.py <path to file>

```
